# Лестница Союза (Union Ladder)

Интерактивный психологический тест, помогающий определить уровень развития отношений по 12-ступенчатой модели.

## Описание проекта

Лестница Союза - это одностраничное веб-приложение, которое позволяет пользователям:

- Пройти психологический тест из 24 вопросов, разделенных на 2 блока
- Получить подробный анализ уровня личной зрелости и отношений
- Визуализировать результаты с помощью наглядной инфографики (кривая Гаусса)
- Получить персонализированные рекомендации
- Скачать результаты теста в формате PDF

## Технологии

- React 18
- TypeScript
- Vite
- TailwindCSS для стилизации
- Framer Motion для анимаций
- Recharts для визуализации графиков
- jsPDF для генерации PDF-отчетов
- React Router для навигации

## Установка и запуск

### Предварительные требования

- Node.js (v16+)
- npm или yarn

### Шаги установки

1. Клонируйте репозиторий
```bash
git clone https://github.com/your-username/12union-test.git
cd 12union-test
```

2. Установите зависимости
```bash
npm install
# или
yarn install
```

3. Запустите проект в режиме разработки
```bash
npm run dev
# или
yarn dev
```

4. Откройте браузер и перейдите по адресу http://localhost:5173

## Сборка и деплой

### Локальная сборка

```bash
npm run build
# или
yarn build
```

После сборки все файлы будут находиться в директории `dist`, которую можно загрузить на любой статический хостинг.

### Деплой на Netlify

Мы подготовили скрипт для удобного деплоя проекта на Netlify:

```bash
# Сделайте скрипт исполняемым (если еще не сделано)
chmod +x deploy.sh

# Запустите скрипт деплоя
./deploy.sh
```

Скрипт создаст архив `12union-netlify.zip`, который вы можете загрузить на Netlify через ручной деплой:

1. Зайдите на [Netlify](https://app.netlify.com)
2. Нажмите 'Add new site' > 'Deploy manually'
3. Перетащите файл `12union-netlify.zip`

Готово! Ваш сайт будет доступен по адресу, предоставленному Netlify (например, `your-site-name.netlify.app`). Вы можете настроить собственный домен в настройках Netlify.

## Структура проекта

- `/src/components` - React компоненты
- `/src/pages` - Страницы приложения
- `/src/utils` - Утилиты, включая алгоритм расчета результатов
- `/src/assets` - Статические ресурсы

## Архитектура приложения

Приложение следует модульной архитектуре с разделением на следующие ключевые модули:

1. **Модуль теста** - управляет отображением вопросов и сбором ответов
2. **Модуль расчета результатов** - анализирует ответы и формирует профиль пользователя
3. **Модуль визуализации** - отображает результаты в виде графиков и инфографики
4. **Модуль экспорта** - генерирует PDF-отчеты с результатами
